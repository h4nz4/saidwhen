## 1. RECOMMENDATION

Ship the minimal version: **stdlib `csv` module, one authenticated "Export my tasks" endpoint (or CLI command) that streams the current user's tasks as a CSV file synchronously.** No new dependency, no background-job queue, no cloud storage, no scheduled/recurring exports.

Rationale tied to recorded constraints:
- **Solo maintainer** (`knowledge/constraints/solo-maintainer.md`) — "every design choice must minimize operational surface." That rules out an async export pipeline, a job table, or an S3 bucket to babysit. Synchronous streaming from the request has zero standing operational surface.
- **External freelancers, individual accounts** (`knowledge/constraints/external-users.md`) — data is per-user and volumes are small (one freelancer's own task list), so a synchronous in-request export won't hit size limits that would force a job queue. Export must be scoped to the authenticated user's own tasks only.

`csv` is in the Python stdlib — nothing to add. Skipped: async jobs / storage / scheduling — add only if a real user is exporting enough rows to time out a request (not plausible for a solo freelancer's task list).

## 3. QUESTIONS (for the PM/owner)

These are genuinely unanswered by the wiki:

1. **Which fields, and in what order?** The wiki doesn't record the task schema. Do you want all columns (id, title, status, due date, timestamps, notes) or a curated subset?
2. **One flat file, or scoped/filtered?** All of a user's tasks in one CSV, or export-what's-currently-filtered (by status/date range)?
3. **Delivery surface** — is TaskLite web-app, CLI, or both? Determines whether this is an HTTP download endpoint, a CLI subcommand, or both.
4. **Excel compatibility needed?** If users open these in Excel, that forces UTF‑8 BOM + CRLF choices (still stdlib, but changes 2 lines). Otherwise plain UTF‑8.
5. **Is import (round-trip) ever coming?** If yes, I'd pin column headers to stable machine keys now; if export is purely human-facing, header labels can be friendly. Cheap to decide now, annoying to change later.
