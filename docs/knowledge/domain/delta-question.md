---
type: Term
title: Delta question
description: The only question an agent asks about a recorded decision with aged evidence.
timestamp: 2026-07-08T20:30:00Z
---

The **delta question** is the prescribed move when a task touches an
accepted Decision whose evidence may be stale: *"this was decided on
\<date> because \<evidence> — is that still true?"* It replaces both
failure modes: re-litigating settled decisions and blindly obeying stale
ones. It is the behavior the name *saidwhen* refers to.
