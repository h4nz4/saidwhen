# TaskLite (example project)

A fictional lightweight task tracker for freelancers, used to demonstrate the
saidwhen convention (v1.0). Browse [knowledge/](knowledge/) and
[openspec/specs/](openspec/specs/) — every requirement links to the decision
that created it (`([why](...))` in the requirement's prose), and every
decision links to the attributed facts — who said it, when — that justified
it. The spec uses the OpenSpec `Requirement`/`Scenario` grammar; why-links
sit inside the requirement text so OpenSpec's delta→main sync carries them
verbatim.
