# Accounts

## Requirements

### ACC-1: Single-user accounts

Each account MUST belong to exactly one user. There is no sharing, no roles,
and no team or organization concept.
([why](../../knowledge/decisions/single-user-accounts.md))

### ACC-2: Data isolation

A user MUST only ever see their own tasks and projects.
([why](../../knowledge/decisions/single-user-accounts.md))
